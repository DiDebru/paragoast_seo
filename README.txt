# Yoast SEO for Drupal 8
Improve your Drupal SEO: write better content using Yoast SEO for Drupal module.
https://www.drupal.org/project/yoast_seo

# How to uninstall
Before uninstalling Yoast SEO for Drupal 8, you need to prepare the module to be uninstalled.
To do so, execute this command in your console, at the drupal root : drush ypu
